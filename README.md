# Automotive RAG System

AI-powered automotive knowledge assistant built with FastAPI and modern AI technologies.

## Features
- Query automotive knowledge base
- Upload and process automotive documents
- RESTful API with automatic documentation
- Docker containerization

## Quick Start

### Local Development
```bash
# Clone repository
git clone <your-repo-url>
cd automotive-rag

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Mac/Linux
# or venv\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt

# Run the application
python app/main.py
